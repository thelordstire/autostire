# 🏆 لوردستاير: الثورة الرقمية والأمنية المطلقة في عالم إدارة قطع الغيار وكماليات السيارات 🚗⚡

---

## ✨ بيان القوة التشغيلية والمستقبلية (The Vision)
أهلاً بكم في المستقبل! ليس مجرد برنامج محاسبة عادي، بل هو **منظومة رقمية خماسية الأنوية** تجمع بين **قوة التوثيق غير القابل للاختراق (Blockchain Security)**، و**الذكاء الهندسي الرياضي لحساب وتفصيل المواد الحرة**، و**الرقابة الفورية اللحظية على نظام تشغيل الجهاز (OS-Level Audit)**. 

لقد وُلد نظام **لوردستاير (Lordstair Auto Parts System)** لينهي تماماً عصر الفوضى المالية، وهدر الخامات غير المحسوب، والتلاعب بالثغرات المحاسبية، مقدماً للمستثمرين وأصحاب القرار أداة تضمن أعلى عوائد ربحية وأمان تشغيلي كامل بنسبة 100%.

## 🛠️ الهندسة الهيكلية الثورية: نظام الأنوية الخمسة النشطة 🚀

يعمل البرنامج بنظام ترابطي متناغم ومنفصل كلياً في العمليات والواجهات، مبني بالكامل على فلسفة **البرمجة الشيئية (OOP)** حيث يتم مراقبة وتتبع كل ذرة حركة برمجياً:

```
                  ┌──────────────────────────────────────────────┐
                  │          💎 واجهة المستخدم الفخمة            │
                  │             (نواة الواجهة)                   │
                  └──────────────────────┬───────────────────────┘
                                         │ (طلبات وإشارات مشفرة)
                  ┌──────────────────────▼───────────────────────┐
                  │           🔌 منسق وموجه الاتصال              │
                  │            (نواة التنسيق والاتصال)            │
                  └──────────────┬───────────────────────┬───────┘
                                 │                       │
         ┌───────────────────────┴────────┐     ┌────────┴───────────────────────┐
         │                                │     │                                │
┌────────▼────────────────────────┐ ┌──────▼─────▼──────────────────────┐ ┌────────▼────────────────────────┐
│         🛡️ نواة الأمن            │ │        ⚙️ نواة العمليات          │ │        💾 نواة البيانات         │
│ (سلسلة الكتل وبلوكات التوثيق)  │ │      (محرك الباكيند الشيئي)       │ │  (قواعد البيانات والاستيراد)    │
└─────────────────────────────────┘ └───────────────────────────────────┘ └─────────────────────────────────┘
```

### 1️⃣ 🛡️ نواة الأمن السيبراني والبلوكتشاين (Security Core)
* **سيرفر الكتل والتوثيق (Blockchain Server):** وداعاً للتلاعب والدفاتر المزدوجة! كل كاشير يسجل علية حدث، أو يعدل مخزوناً، أو يصدر فاتورة، يتم حفر عمليته فوراً داخل كتلة مشفرة بـ **SHA-256** ومربوطة بالكتلة السابقة مع إتمام عملية تنقيب حقيقية (**Proof of Work**). أي تلاعب يدوي مباشر في قاعدة البيانات سيكسر السلسلة فوراً ويطلق جرس الإنذار الأمني!
* **سيرفر الرقابة على نظام التشغيل (OS-Level Audit via psutil):** النظام يراقب حتى عتاد الجهاز! بلحظة الإجراء، يلتقط النظام استهلاك المعالج (CPU %)، استهلاك الذاكرة (RAM %)، والعمليات النشطة بالجهاز لتوثيقها وحمايتها سيبرانياً.
* **تشفير البيانات اللامركزي (Military-Grade Cryptography):** يتم تشفير الهواتف وأسعار الجملة وهوامش الربح الحقيقية عبر معيار التشفير المتماثل الآمن (**Fernet AES**)، فلا أحد يعلم سر صنعتك غيرك!

### 2️⃣ ✂️ أداة التفصيل والتشغيل الديناميكي (Dynamic Fabrication Tool)
تلك المعضلة الأزلية التي تواجه كافة المحلات (فقدان أمتار التظليل، جلود التنجيد، وضفائر الأسلاك، وتقدير الهدر والقص عشوائياً) تم حلها برياضيات ذكية وحرية تامة:
* لا نلزمك بقوالب قياسية جامدة! بل نمنحك **لوحة تفصيل ديناميكية كاملة**.
* أدخل نوع الخامة، ثم اكتب اسم المشروع بحرية تامة (مثال: *"تنجيد مقاعد جلد ألماني فاخر لسيارة مرسيدس E200"*).
* حدد المقاييس الصافية، والنسبة المئوية الحقيقية المتوقعة للهدر والقص، وقيمة المصنعية وشغل يد الفني، وتكلفة المواد المساعدة الأخرى (كالغراء أو المسامير).
* يقوم المحرك الهندسي بحساب التكلفة الإجمالية وحسم الأمتار الفعلية شاملة الهدر من بكرات المستودع بالكسور العشرية، مع تزويدك برسم بياني فوري يحلل نسبة المادة الصالحة مقارنة بالمادة المهدورة.

### 3️⃣ ⚙️ نواة العمليات (Operations Core)
* المحرك الذكي الموجه للكائنات النشطة والشيئية.
* يمنع المبيعات العشوائية، يحسب الورديات بدقة، ويصدر تنبيهات حية فور نزول أي قطعة غيار تحت حد الطلب الحرج لتفادي نفاد المنتجات.

### 4️⃣ 💾 نواة البيانات والانتقال السلس (Data Core)
* خزان البيانات الأمين المدعوم بـ **SQLite**.
* **محرك الهجرة والترقية الخارق:** يمكنك الانتقال من أي برنامج قديم عتيق بضغطة زر! قم برفع ملف الـ **CSV** وسيقوم المترجم الذكي بمطابقة الأعمدة وتشفير أسعارها وحفظها في ثوانٍ.
* تصدير فوري لقاعدة البيانات وصيغ الجرد الفاخرة لـ **Excel** و **CSV**.

### 5️⃣ 🔌 نواة التنسيق والاتصال (Connection Core)
* المايسترو ومنظم الاتصال الذي يضمن الفصل المطلق بين الواجهات والعمليات، ويمنع أي اتصال بالداتابيز دون المرور بالحراسة الأمنية والتوثيق في البلوكتشاين.

## 💎 دليل تشغيل الإثارة والمبيعات للمستخدمين غير التقنيين 💡

تم تصميم الواجهة لتكون في غاية السهولة، البساطة، والروعة البصرية، ليعمل عليها أي موظف فوراً دون تدريب مسبق:

### 🌟 الخطوة 1: الدخول الآمن وبدء الوردية ⏱️
1. أدخل بيانات حسابك (الحساب التأسيسي: `admin` / `admin123`).
2. انطلق إلى صفحة **"الموظفين والورديات"** وافتح ورديتك اليومية بضخ الكاش التأسيسي المتوفر في الدرج. **(بدون وردية مفتوحة، تظل خزائن مبيعات النظام مقفلة أمنياً!)**

### 🌟 الخطوة 2: البيع السريع وتفصيل الخامات 📐
1. **للكماليات وقطع الغيار العادية:** حدد الصنف، أدخل الكمية، واضغط **"إضافة للسلة"**.
2. **لخياطة الجلود وتفصيل الفيميه والضفائر:** اختر الخامة، واكتب اسم وتفاصيل مشروعك الفني، وحدد طول الأمتار الصافية ونسبة الهدر المتوقعة وأجرة الفني، وسيقوم النظام بحساب السعر الأخير تلقائياً وعرض رسم بياني تحليلي ممتع، ثم اضغط **"إضافة عملية التفصيل للسلة"**.
3. حدد الخصم وطريقة دفع الزبون (نقدي، شبكة، تحويل)، ثم اضغط **"إصدار الفاتورة وتحديث المخازن"** ليشهد النظام ولادة **وصل حراري مصمم بشكل يحاكي طابعات المحلات الفخمة** للطباعة والتسليم!

### 🌟 الخطوة 3: التدقيق الجنائي الأمني وإحصائيات المبيعات ⛓️
انتقل لصفحة **"نظام الأمان والكتل"** لتستعرض كتل البلوكتشاين المحفورة بلون ذهبي فخم، وتتحقق من استهلاك معالج وذاكرة الجهاز لحظة البيع، مع مراقبة أداء الخادم السيبراني ومؤشرات استقرار الموارد المكتشفة حياً ومباشرة!

## ⚡ كيف تبدأ تشغيل الثورة الرقمية الآن؟ 🚀

### 📦 متطلبات التشغيل الفورية:
لدينا بيئة بايثون جاهزة ومثبت عليها كافة الملحقات المحترفة المدرجة في ملف الملحقات المرفق `requirements.txt`:
* **Streamlit** (للواجهة الفخمة السريعة)
* **Pandas & Openpyxl** (لقراءة وتصدير وحساب الجداول والإكسسل الملون)
* **Cryptography** (للتشفير العسكري والسيبراني)
* **Psutil** (لمراقبة نظام تشغيل الجهاز وموارده الحية)

### 🏃‍♂️ أمر التشغيل الفوري:
افتح نافذة الأوامر ونفذ الكود السحري:
```bash
cd /home/user/lordstair_auto_parts
streamlit run main.py
```
